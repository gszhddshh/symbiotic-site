About Symbiotic

Symbiotic is a permissionless restaking protocol designed to enhance blockchain security and efficiency. By enabling shared security, it allows staked assets like ETH to secure multiple protocols simultaneously.

With a modular architecture, Symbiotic empowers developers to create secure, scalable, and decentralized ecosystems tailored to their needs.

Learn About Restaking

Symbiotic introduces restaking, a mechanism that reuses staked assets to secure multiple protocols, reducing costs and enhancing interoperability. Its shared security model optimizes capital efficiency for decentralized networks.

Shared Security

Pools resources to reduce staking costs across protocols.

Restaking
